Introduction

The Middle Earth Java application was designed to allow students who use the web application to pull data from the web application and to allow them to make edits or create their own articles to push to the website from a convenient Java application.

The application was developed after the API was implemented in the Laravel web application. An Ant build script also helps users to run the application.

Initial Ideas

Since I was new to frameworks at the time, I decided to opt for Laravel since the backend seemed fairly straightforward. Also, since this was a team project, I ended up doing all of the programming (excluding some CSS changes).

I knew that there would need to be User models, Article models and even Comment models to tie everything together. Also, learning how to get the site hosted on Heroku for the first time was a challenge.

Final Concept

The website is fairly fleshed-out in terms of functionality, but I would like to implement a better overall design to the site, especially in regards to color. The two main colors used would be green, black, white, and a little bit of salmon, but the green is too sparse and there should be other colors for accents.
